h.264

영상 파일 포맷의 종류 - BMP, JPEG, RAW, GIF, PSD, TIFF 등

BMP, JPEG 파일 포맷은 영상의 색상 정보, 해상도 등을 알 수 있는 header 정보가 있음.

RAW 파일 포맷은 header 정보 없이 data만으로 구성.

quantization - 픽셀의 밝기나 색상값을 몇단계 양자화된 표본값으로 결정 (quantization bits)
